The initial template of the Vite+Vue project has integrated Vuex, VueRouter, Axios and other major plug-ins.

> This project is purely used as a template without additional design.

#### Technology stack

- Vite3
- Vue3
- Vuex
- Vue Router
- Axios
- element-plus
- vueuse
- js-cookie
- normalize.css
- eslint + prettier

  ……

#### Common commands

- Install

```
npm install
```

- Start up

```
npm run dev
```

- Compile

```
npm run build
```

- Format code

```
npm run format
```

- Production

```
npm run prod
```
